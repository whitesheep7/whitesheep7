表单的应用

  • 隐藏性

如在密码的 name 后添加 hidden，则密码框就不见了，也有一些好处，我们可以通过隐藏的提交默认值，如填写密码默认 123456，默认不显示，但会提交

  • 只读

在默认的后面添加属性

如 value 后添加 readonly 则，不能修改，只能读取

  • 禁用

在默认的后面添加属性

如 checked 后添加 disable 则，只能选择，在多选中起作用

如按钮禁用的话就不能点击了
